# Rhythm: Weekly 1:1 with Anand Narayanan

**Frequency:** Weekly (every Thursday)
**Where:** Confluence page under your personal space
**Format:** Eisenhower matrix — Important/Urgent quadrants + FYIs

## What to do each week

1. Create a new Confluence page titled `1:1 - Anand - <your name> - YYYYMMDD` as a child of your 1:1 parent page
2. Use the Eisenhower matrix format (see template below)
3. Tag items with status labels: `Important note`, `Need Input`, `FYI`
4. Link to relevant docs, Looms, Jira tickets — Anand likes context he can review async
5. Include a personal/fun section — he values human connection

## Template structure

The page uses a table with four quadrants:

| | |
|---|---|
| **Important but not urgent** | **Important and urgent** |
| Strategic items, upcoming risks, things to spar on | Blockers, decisions needed now, escalations |
| **FYIs and good-to-knows (and fun stuff)** | **Urgent but not important** |
| Status updates, wins, personal stuff | Admin items, approvals, quick asks |

## Tips for talking to Anand (from his User Manual)

- **Lead with data but bring the anecdote.** He loves data but believes anecdotes reveal what data misses.
- **Think long term, execute short term.** Don't pitch something just because one customer wants it — show how it fits the bigger picture.
- **Be direct.** He values simple, honest views. Don't hedge.
- **Be passionate.** He feeds off energy and feels let down by low-passion interactions.
- **Give feedback directly.** He prefers in-person or Slack, not buried in docs.
- **Respect his hours.** 7am–5pm IST + 8:30pm–10pm IST.
- **Fun topics:** Football (European), fitness/workouts, edtech, LOTR, travel, music.
