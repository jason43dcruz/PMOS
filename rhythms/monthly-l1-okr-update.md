# Rhythm: Monthly L1 OKR Update

**Frequency:** Monthly
**Where:** [FY26 O2KR3 Scoring Working Page](https://hello.atlassian.net/wiki/spaces/ITSOL/pages/5677624041/FY26+O2KR3+Scoring+Working+Page)
**What:** Update the L1 OKR scoring page with latest progress, data, and commentary.

## Data Sources

Pull updates from these Atlas projects before updating the OKR page:

- [ATLAS-117333](https://home.atlassian.com/o/2346a038-3c8c-498b-a79b-e7847859868d/s/a436116f-02ce-4520-8fbb-7301462a1674/project/ATLAS-117333)
- [ATLAS-115367](https://home.atlassian.com/o/2346a038-3c8c-498b-a79b-e7847859868d/s/a436116f-02ce-4520-8fbb-7301462a1674/project/ATLAS-115367)

## What to do each month

1. Read the latest updates from the Atlas projects above
2. Read the current state of the [FY26 O2KR3 Scoring Working Page](https://hello.atlassian.net/wiki/spaces/ITSOL/pages/5677624041/FY26+O2KR3+Scoring+Working+Page) via Confluence
3. Gather latest data — metrics, milestones hit, blockers from the Atlas projects
4. Update the page with current scoring and commentary
5. Flag any risks or changes in trajectory

## Tips

- Tie updates back to GOALS.md — especially P0 (Service Collection Uplift) and P1 (1-year investment plan)
- Be honest about what's on track and what's not. No sugarcoating.
- Include links to supporting data (Jira, dashboards, docs) so reviewers can dig in
- If the score has changed, explain why in plain language
