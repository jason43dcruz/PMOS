# Weekly Review Rhythm

<!-- Run this at the end of each week. Say "help me with my weekly review" to the agent. -->

## Checklist

- [ ] Process BACKLOG.md (say "clear my backlog")
- [ ] Review GOALS.md — still accurate?
- [ ] Scan Tasks/ — update statuses, close completed
- [ ] Ask "what should I work on this week?" for next week's focus
- [ ] Add any new Knowledge/ docs from the week
- [ ] **Kernel evolution check:** Scan this week's session-log entries for patterns that should become permanent config. If the same instruction was given 3+ times, graduate it to AGENTS.md or a skill file.
- [ ] **Memory hygiene:** Check session-log for anything that could be found by reading Confluence/Jira/Atlas. If yes, remove it — only keep things that exist exclusively in conversation.
