# Skill: Data to Impact Narrative

**Trigger:** "Build impact narrative," "Prove this matters," "Turn this data into a story"

When the user asks you to convert metrics into a compelling narrative:

1. Structure as: **Headline → Context → Evidence table → So what → What's next**
2. Calibrate framing for the audience:
   - Leadership: Big picture, strategic implications
   - Strategy docs: Deeper analysis, methodology
   - Promotion cases: Quantified impact, ownership
3. Pull specific numbers. No vague claims.
4. End with clear next steps or recommendations

The output should make the data tell a story, not just report numbers.

5. Include a **Data Artifacts** link block (notebook + dashboard) per [Analysis Artifacts](./analysis-artifacts.md). Every narrative should point back to the rerunnable queries and live dashboard.
